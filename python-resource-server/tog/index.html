Usage Examples

# Search TOGAF vault
curl -i "https://api.nzrtnetwork.com/app/togaf/search?q=stakeholder"
# → 402 Payment Required (unpaid)
# → 200 OK + results (after x402 payment)

# Fetch a phase document
curl -i "https://api.nzrtnetwork.com/app/togaf/phase?n=2&doc=Architecture+Vision"
# → 402 Payment Required (unpaid)
# → 200 OK + document content (after x402 payment)

x402-compatible clients (Claude Desktop MCP server) handle payment automatically. Contact nathan@nzrtnetwork.com for MCP server access.
